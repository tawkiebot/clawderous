# Getting Started

## Prerequisites

- OpenClaw account
- AgentMail inbox (or any IMAP provider)
- Telegram bot (optional)

## Setup

1. Install Clawderous via ClawHub
2. Configure your email credentials
3. Start the agent: `npm run dev`
4. Send your first command

## First Command

To: clawderous@agentmail.to
Subject: /memo Hello World

Body: This is my first Clawderous memo!
