# API Reference

## Email Commands

### /memo
Save a quick note to your knowledge base.

**Syntax:**
```
To: clawderous@agentmail.to
Subject: /memo Note Title

Body: Note content here
```

### /blog
Create a blog post.

**Syntax:**
```
To: clawderous@agentmail.to
Subject: /blog Post Title

Body: Post content here
```

### /run
Execute a workflow.

**Syntax:**
```
To: clawderous@agentmail.to
Subject: /run workflow-name

Body: Optional parameters
```

### /reply
Send a reply to a specific email address.

**Syntax:**
```
To: clawderous@agentmail.to
Subject: /reply person@example.com

Body: Reply content here
```
