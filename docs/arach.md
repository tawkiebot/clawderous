Helping friends with their startups. Focused on **agentic development workflows**. Currently using **Wispr + Claude Code**.

## Philosophy

Arach builds tools that:
- Focus on **agentic workflows** over human-only tools
- Prioritize **developer experience** and **automation**
- Create **unified, composable systems** (e.g., SpeakEasy for TTS)
- Value **open source** and **community** (MIT licensed)

## Notable Contributions

### Dewey Pattern
Arach pioneered the **agent-ready documentation** pattern with Dewey, emphasizing:
- Critical context first
- Pattern-based navigation
- Two-layer docs (human + agent versions)
- Scoring systems for agent-readiness

### Agentic Workflows
Focus on making tools that work with AI agents, not just for humans:
- `hooked` - Claude Code improvements
- `scout` - Dictation optimized for agents
- `fabric` - Containerized agent environments
